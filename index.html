Steganography: Message Encryption Inside Images

A project implementing steganography to securely hide encrypted messages within digital images, ensuring both confidentiality and the undetectability of the message. This technique allows sensitive information to be concealed within various image formats with minimal impact on image quality.

RTE-2: Random Timestamp Encryption Using SHA and Unique Hashing

Developed a custom encryption method, RTE-2, which combines SHA hashing with timestamps to generate unique hashes for identical messages. This ensures that each encryption is distinct even when the same data is input, enhancing security by preventing pattern recognition in encrypted messages.

Emergency Communications Tool

Developed a secure messaging app for emergencies, utilizing the RTE-2 custom cryptographic algorithm for encrypted communication. It allows users to broadcast and receive messages on a local network or any IP destination of their choice, scanning and listening on any available port. This tool can be particularly useful during attacks or critical situations, ensuring confidentiality with timestamp-based encryption.

Instant Backup USB (IB-USB)

Created a tool to protect sensitive data during a cybersecurity attack by instantly backing up crucial files from the system onto a USB drive. It also scans and identifies important files based on keywords, ensuring the most valuable data is secured first.

VAPT - Vulnerability Assessment and Penetration Testing Toolkit
